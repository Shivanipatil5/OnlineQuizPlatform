# Online Quiz Platform

A Python and Tkinter-based online quiz application developed as part of an internship task.
The application provides a simple graphical user interface, multiple-choice questions, and automatic score evaluation.

## Features
- Interactive GUI using Tkinter
- Multiple-choice quiz questions
- Automatic score calculation
- Beginner-friendly interface

## Tech Stack
- Python
- Tkinter
- Visual Studio Code

## How to Run
1. Download or clone the repository
2. Run the application using:
   python quiz.py

## Learning Outcomes
- Understood Python GUI development using Tkinter
- Improved logical thinking and event handling
- Gained experience with GitHub version control
